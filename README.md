# Predictive-Analytics-Lab-Exam-2.
## 📌 Overview

This repository contains the implementation of a **classification task** as part of the Predictive Analytics Lab Exam. The project includes Exploratory Data Analysis (EDA), model building, decision boundary visualization, and performance evaluation.

---

## 📁 Repository Structure

```
├── dataset.csv                # Dataset used for analysis
├── Predictive_Analytics_Lab_Exam_2.ipynb   # Main notebook
├── README.md                 # Project documentation
```

---

## 🔍 Tasks Performed

### 1. Exploratory Data Analysis (EDA)

* Checked dataset structure and summary statistics
* Handled missing values (if any)
* Visualized data using plots (scatter plots, histograms, etc.)
* Identified patterns and relationships between variables

---

### 2. Model Building

* Selected a suitable **classification algorithm** (e.g., Logistic Regression / SVM / KNN)
* Split dataset into training and testing sets
* Trained the model using training data

---

### 3. Decision Boundary Visualization

* Plotted decision boundary to understand how the model separates classes
* Visualized dataset points along with classification regions

---

### 4. Model Evaluation

* Evaluated model performance using:

  * Accuracy
  * Confusion Matrix
  * Classification Report (Precision, Recall, F1-score)
